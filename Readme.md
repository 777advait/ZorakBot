# ZorakBot

ZorakBot is the House bot of the Practical Python discord server. Zorak is developed with features by the community. Anyone from our server can contribute. 


## User Commands
#### For-fun commands
- z.hello
- z.catfact
- z.dogfact
- z.pugfact
- z.quote
- z.joke
- z.8ball [question]
- z.taunt
- z.rolldice
- z.catpic
- z.dogpic [breed] (Optional)
- z.pokedex [pokemon]
#### Utility Commands
- z.google [question]
- z.embed [title] [ content ]  
- z.zeus [website]
- z.fakeperson
- z.userinfo/z.whois
- z.avatar/z.av
- z.ping
- z.pipsearch/z.pypi/z.pip [package]
- z.git/z.github [endpoint]
#### Admin Commands
- z.echo [message]
- z.rules [title] [rules]
